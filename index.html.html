개인정보처리방침

본 앱(“학우너”)은 이용자의 개인정보를 직접적으로 수집하거나 저장하지 않습니다.

1. 수집하는 개인정보

앱은 어떤 개인정보도 직접 수집하지 않습니다.
다만 카카오 및 네이버 등 외부 로그인 서비스 이용 시 해당 서비스 제공자가 제공하는 최소한의 인증 정보를 사용할 수 있습니다.

2. 개인정보의 이용 목적

외부 로그인 서비스에서 제공하는 인증 정보는 로그인 기능 제공 외의 다른 목적으로 사용되지 않습니다.

3. 제3자 제공

앱은 이용자의 개인정보를 어떠한 제3자에게도 제공하지 않습니다.

4. 개인정보 보관 및 파기

앱은 자체 서버를 운영하지 않으며 사용자의 정보를 저장하지 않습니다. 따라서 개인정보 보관 및 파기의 대상이 존재하지 않습니다.

5. 외부 서비스 이용

카카오 로그인, 네이버 로그인 사용 시 그 서비스 제공업체의 개인정보처리방침이 적용됩니다.

6. 개인정보처리방침 변경

본 방침은 필요 시 변경될 수 있으며 변경 시 업데이트된 버전이 본 페이지에 게시됩니다.
